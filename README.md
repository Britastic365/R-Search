**About the Extension:**
The extension makes it much faster to search romance.io for authors, books, and series. It skips the step of opening romance.io, typing or pasting in the text, and hitting the "search" button.

**Browser Compatibility:**
Compatible with Chrome, Edge, and Opera. 

**How to Use R Search:**
The extension requires zero setup and is ready to use as soon as it's installed.

  1. Find the text you want to search. For example:  Jane Austin

  2. Select the text you want to search with your cursor.

  3. With the text selected, right click.

  4. In the menu, select: Search romance.io for ' [selected text]'

  5. A new browser tab will open with the romance.io search results.

**Privacy Policy for R Search:**
Effective Date: February 15, 2026

  1. **No Data Collection:** This extension does not collect, store, or transmit any personal information or browsing history.

  2. **How it Works:** The extension only functions when you explicitly right-click a selection of text. It takes that text and opens a new tab with the romance.io search URL. None of your data is recorded or sent to the developer.

  3. **Third-Party Websites:** When you use the "Search romance.io" feature, you are directed to romance.io. This policy does not cover how romance.io handles your data. We recommend reviewing their privacy policy separately.

  4. **Contact:** If you have questions about this extension, you can contact the developer via the Chrome Web Store support link.
